# Section Coverage and Structural Adherence Analysis

This document presents the quantitative analysis of structural adherence across the four evaluated documentation architectures: **Dev-Plan (dev)**, **Autonomous MAS (mas)**, **Single-Agent (single)**, and **Original Ground Truth (ground)**. 

The evaluation compares the raw sections generated by each approach against an established taxonomy of 12 standard README sections (e.g., Installation, Usage, Contributing).

## 1. Aggregated Coverage Metrics (N=20 Repositories)

| Metric | Dev-Plan (dev) | MAS (mas) | Single-Agent (single) | Ground Truth (ground) |
| :--- | :---: | :---: | :---: | :---: |
| **Total Generated Sections (P)** | 209 | 170 | 152 | 221 |
| **Total Matched Sections (TP)** | 180 | 167 | 118 | 160 |
| **Avg. Outputted Sections (P/N)** | 10.45 | 8.50 | 7.60 | 11.05 |
| **Avg. Covered Sections (TP/N)** | 9.00 | 8.35 | 5.90 | 8.00 |
| **Section Coverage Recall** | **75.00%** | 69.58% | 49.16% | 66.66% |
| **Precision (TP/P)** | 86.12% | **98.23%** | 77.63% | 72.39% |

*(Note: Recall = (TP/N) / 12 standard sections. Precision = TP / P. Total Repositories, N = 20)*
